## ADDED Requirements

### Requirement: Central Navigation Hub
The system SHALL provide a central landing page (Home) that acts as a navigation hub directing users to the three main business units: Logistics, Industry, and Distribution.

#### Scenario: User visits homepage
- **WHEN** a user lands on the website root URL
- **THEN** they see distinct, prominent sections or pathways for "Operador Logístico", "Indústria", and "Distribuição"

### Requirement: Corporate Information
The system SHALL present institutional information about Suppra Foods, including its mission ("a ponte segura até a mesa do consumidor"), history, and corporate values.

#### Scenario: User checks company background
- **WHEN** a user navigates to the "Sobre Nós" section
- **THEN** they can read the company's history, mission, and view the corporate infrastructure
