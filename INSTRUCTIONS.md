# Instruções do Projeto - Suppra Foods

> [!IMPORTANT]
> Independente de qual desenvolvedor ou agente, precisa consultar primeiro as 'Instructions' antes de executar qualquer tarefa dentro do projeto e a cada novo acesso ao projeto.

## Diretrizes de Desenvolvimento

1. **Estética Premium**: O design deve ser visualmente impactante, utilizando a paleta de cores institucional (Navy e Azul Suppra) com gradientes modernos e sombras suaves.
2. **Modularização**: Componentes de UI (Hero, Stats, Cards, FAQ) devem ser logicamente separados e reutilizáveis.
3. **Responsividade**: Prioridade total para dispositivos móveis, garantindo que a experiência de conversão seja fluida em telas pequenas.
4. **Ativos Visuais**: Não utilizar placeholders. Todas as imagens devem ser de alta qualidade e relacionadas ao contexto de logística de cadeia fria (câmaras frias, caminhões refrigerados, etc).
5. **Foco em Conversão**: O objetivo principal é a captura de leads. Botões de CTA e o Modal de Cotação devem ser proeminentes e funcionais.
6. **Performance**: Otimizar o carregamento de imagens e scripts para garantir uma landing page rápida.
